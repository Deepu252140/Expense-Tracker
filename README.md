# Expense Tracker

A simple Expense Tracker web app built using Flask.

## Features
- Add new expenses
- View all expenses
- Filter by category
- Delete expenses
- Basic analytics

## Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000/
